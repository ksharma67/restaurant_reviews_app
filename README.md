<h2><a id="user-content-run-the-application" class="anchor" aria-hidden="true" href="#run-the-application"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Run the Application</h2>
<p>In order to run the application you have some options:</p>
<ol>
<li>
<p>Open it <a href="https://stearruda.github.io/restaurant-reviews-app-stage-1/" rel="nofollow">here</a></p>
</li>
<li>
<p>Run it localy</p>
</li>
</ol>
<ul>
<li>
<p>Download as .zip file or clone this project:</p>
<pre><code>$ git clone https://github.com/stearruda/restaurant-reviews-app-stage-1.git
</code></pre>
</li>
<li>
<p>In this folder, start up a simple HTTP server to serve up the site files on your local computer. Python has some simple tools to do this, and you don't even need to know Python. For most people, it's already installed on your computer.</p>
</li>
<li>
<p>In a terminal, check the version of Python you have: <code>python -V</code>. If you have Python 2.x, spin up the server with <code>python -m SimpleHTTPServer 8000</code> (or some other port, if port 8000 is already in use.) For Python 3.x, you can use <code>python3 -m http.server 8000</code>. If you don't have Python installed, navigate to Python's <a href="https://www.python.org/" rel="nofollow">website</a> to download and install the software.</p>
</li>
<li>
<p>With your server running, visit the site: <code>http://localhost:8000</code>.</p>
</li>
</ul>
